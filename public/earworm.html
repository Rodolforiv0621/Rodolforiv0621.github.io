Earworm Project

Earworm is a web app aimed at connecting individuals and artists with like minded individuals based on criteria such as music taste, sex, age, and location.

Earworm is a senior capstone project for the California State University of Northridge. The project was developed from August 2022 to May 2023.

About

Earworm was inspired by the idea of using social media to find people who are similar to you. The project includes features such as adding a bio, changing a bio, adding and changing a profile picture, reducing or increasing matching distance which is used by calculating distance based on zipcode which gives you an approximate distance. The project also allows you to send and receive messsages directly to another user that you matched with. All these features use data that is stored on a database on firebase using javascript and firebase api.
